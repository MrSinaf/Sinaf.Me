# Mon site web personnel (/▽＼)

Vous pouvez observer ce projet en >action< ici : [https://sinaf.me/](https://sinaf.me/)

Étant un projet purement personnel, il ne cherche pas à être accessible. Il peut tout de même rester un bon moyen pour vous d'observer ce qui se passe sous sa *jupe* (✿◠‿◠).
Autant pour vous permettre d'apprendre, que pour vous y inspirer!

**Développé entièrement en C# avec Blazor .Net10, hébergé sous Linux (Debian13)**

```ini
# Variables d'environnement
DB_WARHAMMER
DB_WEB
PASS
```
